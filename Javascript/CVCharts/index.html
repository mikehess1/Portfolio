CV Charts

This tool provides visual representation of statewide data for Maryland, Florida, Texas and California regarding the COVID-19 virus. Charts are updated daily at 4pm ET and data is sourced from https://covidtracking.com. Individual states can be excluded by clicking state abbreviations at the top of each chart. Best viewed on a computer, tablet or landscape mobile.

Please submit any feedback and feature requests to hello@mikehess.net. Check back for updates as more charts will be added in the coming days. Be safe, Stay healthy; Wishing everyone the best!

Total Confirmed Cases

This graph shows the total number of COVID-19 confirmed cases per state over time. Ideal scenario occurs when the lines flatten out.

Cases by % Population

This graph shows the percent of each state's population testing positive for COVID-19.

Daily New Cases

This graph shows the number of new COVID-19 confirmed cases each day per state. Ideal scenario occurs when the bars fall toward zero.
