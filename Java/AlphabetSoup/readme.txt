This Java program finds specified words within a word search puzzle.  The program accepts an input file containing the dimensions of the word search, the letters within the word search and the words to be found within the word search puzzle.  The program outputs each word with the index coordinates of the first and last letters.  The program assumes no errors in the input file.

Example input:
5x5
H A S D F
G E Y B H
J K L Z X
C V B L N
G O O D O
HELLO
GOOD
BYE


Example output:
HELLO 0:0 4:4
GOOD 4:0 4:3
BYE 1:3 1:1
