This program accepts user input which should be a 16 or 32 byte (32 or 64 character) hexadecimal string. The program will print out the corresponsing 12 or 24 BIP39 mnemonic seed phrase, or an error in the event of invalid user input.  This program utilizes the embit library which can be installed via CLI using the following command:

pip3 install embit

 For more information, visit https://embit.rocks
